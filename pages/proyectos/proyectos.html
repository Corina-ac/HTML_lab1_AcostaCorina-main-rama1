Proyectos Académicos

🌐 Portafolio Web Personal

Desarrollé una página web personal responsiva utilizando HTML5. Este proyecto incluye múltiples secciones como perfil profesional, multimedia, horario académico y formularios de contacto.

Tecnologías: HTML5, CSS3, Navegación entre páginas

Aprendizajes Clave:

  • Estructura semántica de HTML5
  • Navegación efectiva entre páginas
  • Uso correcto de etiquetas y atributos
  • Formularios con validación

📋 Formularios Interactivos HTML5

Implementé varios formularios con diferentes tipos de entrada: búsqueda, registro, contacto, login y selección de opciones. Incluye validación de datos y patrones regex para garantizar la integridad de la información.

Características: Validación de contraseña, casillas de verificación, selectores de fecha/hora, campos de búsqueda avanzada

🎨 Diseño Visual y Estilos CSS

Aplicación de estilos modernos incluyendo centrado de texto, tipografía personalizada, esquema de colores institucional, márgenes y separaciones visuales, y una experiencia de usuario mejorada con efectos de hover y transiciones.

Componentes: Navegación responsiva, tablas estilizadas, cards, gradientes, sombras y animaciones suaves
